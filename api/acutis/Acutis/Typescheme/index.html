Module Acutis.Typescheme

Defines the types (integer, record, etc.).

  • Public API for declaring type schemes.
  • Internal utilities.
module Variant : sig ... end

This is a common structure shared by Enum and Union.

module Enum : sig ... end
module Union : sig ... end
type ty =
| Unknown of Variant.row Stdlib.ref(*

The row is for unification with variant types during destructuring.

*)
| Int
| Float
| String
| Nullable of t
| List of t
| Tuple of t list
| Record of t Map.String.t Stdlib.ref
| Dict of t * Set.String.t Stdlib.ref(*

The set tracks which keys have been used, so they can build pattern-matching decision trees.

*)
| Enum of Enum.t
| Union of string * t Union.t
and t = ty Stdlib.ref

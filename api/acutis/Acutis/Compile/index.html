Module Acutis.Compile

Orchestrate the Lexer, Parser, Typechecker, and Matching to produce the final template.

val parse : fname:string -> Stdlib.Lexing.lexbuf -> Ast.t
  • raises Error.Acutis_error
type escape = Ast.escape =
  1. | No_escape
  2. | Escape
type echo_format = Ast.echo_format =
  1. | Fmt_string
  2. | Fmt_int
  3. | Fmt_float
  4. | Fmt_bool
type echo = Typechecker.echo =
  1. | Echo_var of string
  2. | Echo_string of string
  3. | Echo_field of echo * string
type 'a node =
  1. | Text of string
  2. | Echo of (echo_format * echo) list * echo_format * echo * escape
  3. | Match of 'a eval Data.t array * 'a nodes Matching.t
  4. | Map_list of 'a eval Data.t * 'a nodes Matching.t
  5. | Map_dict of 'a eval Data.t * 'a nodes Matching.t
  6. | Component of 'a * 'a eval Data.t Map.String.t

The names of variables are preserved as strings in the Data.t values.

and 'a eval =
  1. | Var of string
  2. | Block of 'a nodes
  3. | Field of 'a eval Data.t * string
and 'a nodes = 'a node list
type 'a template =
  1. | Src of 'a template nodes
  2. | Fun of Typescheme.t Map.String.t * 'a
type 'a t = {
  1. types : Typescheme.t Map.String.t;
  2. nodes : 'a template nodes;
  3. name : string;
}

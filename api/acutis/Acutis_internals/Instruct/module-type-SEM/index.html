Module type Instruct.SEM

Define the semantics of our abstract language.

  • Statements and expressions.
  • Functions.
  • Control flow.
  • Standard values.
  • Strings and characters.
  • Arrays.
  • Hash tables.
  • Buffers.
  • Promises.
  • Data
  • Importing and exporting.
